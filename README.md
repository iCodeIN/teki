Touhou-style shoot'em up written in Rust using legion and sdl2.

<h3 align="center"><img src="resources/teki.gif" height="400px"></h3>

### How to play (Control)

  * Arrow key : Move left, right, up or down
  * Space key : Shoot a bullet
